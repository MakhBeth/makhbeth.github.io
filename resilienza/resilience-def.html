What is Frontend Resilience?

The ability of a web application to keep working when things go wrong.
  • JavaScript fails to load
  • The network is slow or offline
  • The browser is old or limited
  • An ad-blocker breaks your scripts

~1-3% of page views have no working JavaScript.
Not because users chose so — because something went wrong.
